# Group 6 CSS Layout Webpage Design

## Lab 4: Understanding Layout Control in CSS

### Objective
Develop control over CSS layout by mastering:
- **Box Model** (margin, padding, border)
- **Positioning** (static, relative, absolute, fixed, sticky)
- **Float and Clear**

This project demonstrates proper use of these core CSS layout techniques while intentionally exposing layout issues when concepts are misapplied.

---

### Features Implemented

#### Part 1: Box Model
- Three stacked demonstration boxes with varying padding and margin
- Clear visual distinction between content, padding, border, and margin

#### Part 2: Positioning
- Fixed header that stays at the top during scroll
- Sticky navigation bar
- Relative + Absolute positioning (NEW labels positioned at top-right corners)

#### Part 3 & 4: Float Layout
- Two-column layout using `float: left` and `float: right`
- Proper use of `clear: both` on footer to prevent overlap
- Combined all techniques into one cohesive responsive-looking page

**Constraints Followed:**
- No Flexbox
- No CSS Grid
- Only Box Model, Positioning, and Float used

---

### Technologies Used
- HTML5
- CSS3 (Pure vanilla CSS)
