
# Group no 3- Student Group Portfolio

##  Lab Week 3: HTML Structure & CSS Styling

A clean, modern, and responsive single-page website created as part of **Web Development Lab Week 3**.

---

###  Objective
To apply HTML structure and CSS styling using **Inline, Internal, and External** methods while demonstrating various CSS selectors.

### Requirements Fulfilled

- **HTML Structure**: Semantic sections (Header, Navigation, About, Skills, Daily Routine)
- **CSS Methods Used**:
  - **External CSS** (`styles.css`) – Primary styling
  - **Internal CSS** (`<style>` tag) – Navigation menu and headings
  - **Inline CSS** – Limited use on header elements
- **Selectors Demonstrated**:
  - Element selectors
  - Class selectors
  - ID selector (`#header`)
  - Group selector (`h1, h2, h3`)
  - Descendant selector
  - Pseudo-class (`:hover`)

---

###  Website Sections

- **Header** – Group introduction
- **Sticky Navigation Menu** – Easy navigation between sections
- **About Us** – Group description with photo
- **Skills** – Technical skills list
- **Daily Routine** – Typical day-to-day activities
- **Footer** – Assignment information

---

### Technologies Used

- HTML5
- CSS3 (Flexbox)
- External, Internal & Inline CSS
- Responsive Design

---

###  How to Run

1. Download all three files
2. Ensure they are in the **same folder**:
   - `index.html`
   - `styles.css`
   - `README.md`
3. Open `index.html` in any web browser
